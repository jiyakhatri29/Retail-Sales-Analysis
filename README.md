# 📊 Retail Sales Analysis using Python

## 📌 Project Overview

This project analyzes retail sales data using **Python, Pandas, NumPy, and Matplotlib**. It demonstrates the complete data analysis workflow, including data cleaning, feature engineering, exploratory data analysis (EDA), data visualization, exception handling, and report generation.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

---

## 📂 Features

- Read sales data from a CSV file
- Handle missing values using mean imputation
- Remove duplicate records
- Convert date column into datetime format
- Create new columns:
  - Total Sales
  - Profit Margin
  - Sales Category
- Perform Region-wise Sales Analysis
- Perform Product-wise Sales Analysis
- Perform Category-wise Sales Analysis
- Calculate Monthly Sales
- Perform statistical analysis using NumPy
- Generate visualizations using Matplotlib
- Export cleaned data to a new CSV file
- Handle exceptions using try-except

---

## 📈 Visualizations

The project includes the following charts:

- 📊 Region-wise Sales (Bar Chart)
- 📊 Product-wise Sales (Horizontal Bar Chart)
- 🥧 Category-wise Sales Distribution (Pie Chart)
- 📈 Monthly Sales Trend (Line Chart)
- 📉 Profit Distribution (Histogram)
- 🔵 Units Sold vs Total Sales (Scatter Plot)
